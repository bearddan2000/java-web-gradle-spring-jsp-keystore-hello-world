# java-web-gradle-spring-jsp-keystore-hello-world

## Description
A springboot web app with jsp support with self-signed ssl.

#W Tech stack
- java
  - springboot
  - jsp
- gradle

## Docker stack
- gradle:jdk11
- openjdk:17-alpine

## To run
`sudo ./install.sh -u`
[Available at](https://localhost)

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
